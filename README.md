# parking
Тестовое задание для разработчика
используется фреймворк Laravel

### Установка

Настроить .env на mysql базу данных

```bash
composer install
php artisan migrate
php artisan db:seed
```

Настроить зависимости Javascript и CSS
```bash
npm install
npm run dev
```
Запустить сервер
```bash
php artisan serve
```
